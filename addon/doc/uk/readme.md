# GemVDA — Google Gemini AI для NVDA

## Короткий опис

GemVDA інтегрує можливості Google Gemini AI безпосередньо в NVDA, надаючи незрячим та слабозорим користувачам потужну допомогу штучного інтелекту. Додаток підтримує різні моделі Gemini, включаючи Gemini 3, Gemini 2.5 Pro та Flash для чату, опису зображень, аналізу відео та іншого.

## Функції

* **Чат з ШІ**: спілкуйтеся з Gemini AI безпосередньо з NVDA
* **Опис екрана**: захоплюйте та описуйте весь екран
* **Опис об'єкта**: описуйте поточний об'єкт навігатора
* **Аналіз відео**: записуйте відео з екрана та аналізуйте його за допомогою Gemini
* **Додавання зображень**: додавайте зображення з файлів для опису штучним інтелектом
* **Історія розмов**: зберігайте контекст з декількох повідомленнь
* **Кілька моделей**: вибирайте з різних моделей Gemini відповідно до своїх потреб
* **Налаштування параметрів**: налаштовуйте температуру, токени, потокову передачу та інше

## Вимоги

* NVDA 2023.1 або новіша версія
* Ключ API Google Gemini (доступний безкоштовний тариф)
* Підключення до Інтернету

## Налаштування

### Отримання ключа API

1. Відвідайте [Google AI Studio](https://aistudio.google.com/apikey)
2. Увійдіть за допомогою свого облікового запису Google
3. Створіть новий ключ API
4. Скопіюйте ключ для використання в додатку

### Налаштування API-ключа

1. Натисніть NVDA+N, щоб відкрити меню NVDA.
2. Перейдіть до «Параметри» > «Налаштування».
3. Виберіть категорію «Gemini AI».
4. Натисніть «Налаштувати API-ключ...».
5. Вставте свій API-ключ і натисніть «OK».

## Комбінації клавіш

| Комбінація | Дія |
|----------|--------|
| NVDA+G | Відкрити діалог Gemini AI |
| NVDA+Shift+E | Описати весь екран |
| NVDA+Shift+O | Описати об'єкт навігатора |
| NVDA+V | Почати/зупинити запис відео для аналізу |

## Використання діалогу Gemini

Коли ви відкриваєте діалог Gemini за допомогою NVDA+G:

1. **Модель**: Виберіть модель Gemini
2. **Системна підказка**: Необов’язкові інструкції щодо відповіді Gemini
3. **Історія**: Переглянути історію розмови
4. **Запит**: Введіть своє повідомлення або питання
5. **Відправити**: Надіслати запит Gemini
6. **Додати зображення**: Додати файл зображення для аналізу Gemini
7. **Очистити**: Очистити історію розмови
8. **Копіювати відповідь**: Скопіювати останню відповідь у буфер обміну

### Поради щодо діалогу

* Натисніть Enter у полі запиту, щоб швидко надіслати його.
* Використовуйте клавішу Tab для переходу між елементами керування.
* Історія оновлюється автоматично під час чату.
* Додані зображення надсилаються разом із наступним повідомленням.

## Налаштування

Доступ до налаштувань: Меню NVDA > Параметри > Налаштування > Gemini AI:

* **Початкова модель**: Виберіть бажану модель Gemini
* **Температура (0-200)**: Визначає креативність відповідей (0=точні, 200=креативні)
* **Макс. кількість токенів**: Максимальна довжина відповідей
* **Потокова передача відповідей**: Показувати відповіді у міру надходження
* **Режим розмови**: Включати історію чату для контексту
* **Запам’ятовувати системну підказку**: Зберегти вашу власну підказку
* **Блокувати клавішу Escape**: Запобігати випадковому закриттю діалогу
* **Фільтрувати Markdown**: Видаляти форматування markdown з відповідей

### Звуковий відгук

* **Відтворювати звук при надсиланні запиту**: Звукове підтвердження під час надсилання запиту  
* **Відтворювати звук під час очікування**: Звук перебігу під час обробки AI  
* **Відтворювати звук при отриманні відповіді**: Сповіщення, коли відповідь отримана

## Доступні моделі

* **Gemini 3 Pro (попередній перегляд)**: найпотужніша модель з функціями міркування
* **Gemini 3 Flash (попередній перегляд)**: швидка модель з функціями міркування
* **Gemini 2.5 Pro**: потужна модель, готова для використання у виробництві
* **Gemini 2.5 Flash**: швидка та ефективна для більшості завдань
* **Gemini 2.5 Flash-Lite**: легка та швидка у відповідях
* **Gemini 2.5 Flash Image**: оптимізована для завдань, пов'язаних із зображеннями

## Функції для зображень і відео

### Опис екрана (NVDA+Shift+E)

Захоплює весь екран і надсилає його до Gemini для детального опису. Корисно для:

* Ознайомлення з невідомим інтерфейсом
* Отримання загального уявлення про вміст
* Ідентифікації  елементів, які NVDA не може описати

### Опис об’єкта (NVDA+Shift+O)

Захоплює тільки поточний об'єкт навігатора. Корисно для:

* Опису окремих елементів інтерфейсу
* Визначення зображень або іконок
* Отримання деталей про сфокусовані елементи керування

### Аналіз відео (NVDA+V)

1. Натисніть NVDA+V, щоб розпочати запис.
2. Виконайте дії, які ви хочете проаналізувати.
3. Натисніть NVDA+V ще раз, щоб зупинити запис.
4. Зачекайте, поки Gemini проаналізує відео.

Корисно для:

* Розуміння візуальних робочих процесів
* Отримання покрокових описів
* Аналізу динамічного контенту

## Усунення несправностей

### «Бібліотека Google GenAI не встановлена»

Запустіть інсталятор залежностей:
1. Перейдіть до %APPDATA%\nvda\addons\GemVDA
2. Запустіть install_deps.bat або install_deps.py
3. Перезапустіть NVDA

### «Не налаштовано ключ API»

Налаштуйте ключ API в Налаштуваннях > Gemini AI > Налаштувати ключ API

### Відповіді надто короткі або обрізані

Збільшіть параметр "Максимальна кількість вихідних токенів"

### Відповіді надто випадкові

Зменште параметр Температура (спробуйте значення 50-100)

## Повідомлення про конфіденційність

* Ваші повідомлення та зображення надсилаються до API Gemini від Google.
* Ключі API зберігаються локально у вашій конфігурації NVDA.
* Жодні дані не передаються розробнику додатка.
* Детальнішу інформацію див. у політиці конфіденційності Google щодо штучного інтелекту.
